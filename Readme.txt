1. Install Rad studio 12
2 First enable view extenions:
	In file explorer click on view -> show -> check file name extension
3. Extract this whole folder in your D drive
4. Go to:
	D: -> Crypto Wallet -> GUI Proj
5. Double Click to open Crypto-Wallet.cbproj - project will open
6. To run pre-build .exe , go to " D:\Crypto Wallet\GUI Proj\Win64x\Debug " and double click Crypto-Wallet.exe to run it
